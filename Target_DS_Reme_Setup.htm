Target Data Services Remediation Setup

Use this page to View Data Services Remediation Rule Information.

To access this page:

  1. Access Transform.
  2. Click the Targets icon on the Process Area Launch page.
  3. Click the Data Services Remediation icon for the selected Target.
  4. Click Edit.
