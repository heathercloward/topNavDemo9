Set Org Unit Assignment Security for a Request

If the request is based on a template that has Org Units assigned, the templateâ€™s Org Unit assignments are copied to the request when it is added.

These Org Unit assignments display on the Request (Org Unit Assignments) page and may be changed depending on settings described below.

A setting will control whether only Org Unit values assigned to the requester display or all Org Unit values for the template display on the Request (Org Unit Assignments) page. A setting will also control whether the ASSIGNED check box can be enabled or disabled on this page, which allows a user to add or remove Org Unit values from the request. This setting can control user access to request data based on a userâ€™s assigned Org Unit values.

These parameters are set at the global level and may be overridden at the individual template level.

To navigate to the settings at the global level:

  1. Select Configuration > Parameters from Navigation pane.

  2. Click the Template tab.

    To set security parameters that will be used by the Request (Org Unit Assignments) page:

  3. Click the Only Show Org Unit Values Assigned to Requester check box to enable it.

    NOTE: On the Request (Org Unit Assignments) page, only those Org Unit values assigned to the requester display. If the check box is not selected, Org Unit values assigned to all users on the template display.

  4. Click Edit.

    View the field descriptions for the Parameters page, Template tab

  5. Select an option from the Org Unit Security Level list box.

    NOTE: The value chosen sets how Org Unit values are assigned to requests by default. The ASSIGNED column on the Request (Org Unit Assignments) page will be enabled or disabled based on the Org Unit Security Level. Options are:

    • OrgUnit Values Available For Assignment â€“ All of the individual Org Unit values possible for the request are available for assignment to the request, but no assignments are assigned by default. (None of the ASSIGNED check boxes are enabled by default.)

    • User Assignments Defaulted, Change Allowed â€“ All of the individual Org Unit values available to the user creating the request are assigned to the request by default, but may be unassigned. (All of the ASSIGNED check boxes on the Request (Org Unit Assignments)page are enabled by default.)

    • User Assignments Defaulted, No Change Allowed â€“ All of the individual Org Unit values available to the user are assigned to the request and cannot be unassigned. (All of the ASSIGNED check boxes on the Request (Org Unit Assignments) page are enabled by default.) If this option is selected, the Select All, Unselect All, and Reset Defaults buttons are disabled on the Request (Org Unit Assignments) page.

  6. Click Save.

The Org Unit Security Level set on theParameterspage is the default value for all newly created templates.

To change the global parameter values for an individual template, navigate to the settings at the template level:

  1. Click Team on Navigation pane.
  2. Click Templates for a team.
  3. Click VerticalView for a template.
  4. Click Advanced tab.

  5. Click Edit.

    View the field descriptions for the Templates page, Advanced tab

  6. Make choices as described above.
