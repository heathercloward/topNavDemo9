Create Schedules

Schedules added in Common can be used across all dspMigrateâ„¢ components that have schedules, such as refreshing a source in Collect. Schedules can recur on several intervals, including every few minutes, every Tuesday and Thursday, or every October 19th.

To create a schedule:

  1. Select Tools > Schedules in Navigation pane.

  2. Click Add.

    View the field descriptions for the Schedules page

  3. Enter the name of schedule in Name field.

    NOTE: Schedules must have a unique name.

  4. Enter a description of schedule in Description field.
  5. Select a recurrence schedule from Recurrence ID list box to determine the pattern of the schedule.
  6. Select a time for Black Out Time Start to specify a start time for when the schedule does not apply.
  7. Select a time for Black Out Time End to specify an end time for when the schedule does not apply.
  8. Select blackout days to determine which days the schedule does not run.
  9. Click Save.

  10. Click link for Recurrence ID.

    View the field descriptions for the Schedule â€“ Configure page

    NOTE: The fields available to configure the schedule vary according to the Recurrence ID.

  11. Click Edit.
  12. Populate the recurrence fields using the following table.
  13. Click Save.
